
## 使用说明

**production:** 

检查当前的部署文件内容

`make build`

创建 ./1-deploy 下面的资源

`make apply`

删除 ./1-deploy 下面的资源

`make delete`

查看当前创建好的 pod

`make get`

---
**staging:**

检查当前的部署文件内容

`make build-stg`

创建 ./1-deploy 下面的资源

`make apply-stg`

删除 ./1-deploy 下面的资源

`make delete-stg`

查看当前创建好的 pod

`make get-stg`
